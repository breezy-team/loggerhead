Loggerhead is a web viewer for projects in bazaar. It can be used to navigate a branch history, annotate files, view patches, perform searches, etc. It's heavily based on bazaar-webserve, which is itself based on hgweb for Mercurial. The primary difference is that loggerhead is built on top of TurboGears.

Loggerhead is released under the GPL license, and is written entirely in python 2.4.
